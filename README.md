## Laravel Configuration
```bash
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
$ php artisan optimize:clear
```
## Angular Congfiguration
```bash
$ cd Front/cinema
$ ng install
$ ng serve --open
```
